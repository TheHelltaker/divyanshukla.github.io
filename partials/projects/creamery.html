A&M Creamery


For my application development course, I built an employee and store management system for the fictitious A&M Creamery with full front and back-end functionality. The application allows different kinds of users (employees, managers, and administrators) to manage employees, stores, flavors, employment, shifts, and jobs based on their permissions.

This project was a winner of the Chili Pepper Award, which was awarded to the applications with the best functionality, design, and user experience.
