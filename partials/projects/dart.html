Dart


Dart is an application I built for myself to keep track of the things I think of throughout the day that don't seem important enough to write down until I find myself struggling to remember them later.

Wanting something as simple and accessible as Spotlight search and more conducive to multi-tasking than the standard to-do list app, I built the following desktop application. Hitting enter saves my note to my Backand database, recording the text and the timestamp of creation. I didn't want to display all of my reminders on the app itself as it would be a distraction.

Since I didn't want to see a lot of information on the same application I use to input data, I built a web application connected to the same database to manage all my notes.
